## 🚀 Quick Start

### Install

```bash
git clone https://github.com/maheshBawa/Agent-X.git
cd Agent-X
```

<table>
<tr>
<td>

**macOS / Linux**
```bash
bash setup.sh
```

</td>
<td>

**Windows (PowerShell)**
```powershell
.\setup.ps1
```

</td>
</tr>
</table>

Add to your PATH so you can use `agent-x` from anywhere:
```bash
export PATH="$PATH:/path/to/Agent-X"   # add to ~/.bashrc or ~/.zshrc
```

### Initialize & Build

```bash
mkdir my-saas && cd my-saas
agent-x init    # 🏗️  Sets up Agent-X in your project
claude           # 🤖  I take over from here
```

### Update After Upgrading Agent-X

```bash
cd /path/to/Agent-X && git pull     # 📥  Get latest Agent-X features
cd /path/to/my-saas && agent-x update   # 🔄  Push updates to your project
```

> Updates hooks, quality gates, settings, and templates — without touching your project state, architecture docs, or build progress.

### CLI Commands

| Command | Description |
|:--------|:------------|
| `agent-x init` | 🏗️ Initialize Agent-X in the current directory |
| `agent-x update` | 🔄 Pull latest hooks, settings, and templates into your project |
| `agent-x status` | 📊 Show current project phase and state |
| `agent-x reset [PHASE]` | ⏪ Roll back to a specific phase (e.g., `ARCHITECTURE`) |
| `agent-x version` | 🏷️ Show Agent-X version |

This isn't about replacing human creativity. It's about removing the gap between having an idea and holding the product.
