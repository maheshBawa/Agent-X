# Static Site Project Scaffold

## Directory Structure
```
├── index.html                  # Main page
├── css/
│   └── styles.css              # Stylesheet
├── js/
│   └── main.js                 # JavaScript
├── assets/
│   └── images/                 # Image assets
├── pages/                      # Additional HTML pages
└── .gitignore
```

## Setup
No build tools required. Open `index.html` in a browser to develop.
For Tailwind CSS, add via CDN or set up a build step.

## Deployment
Upload files to any static hosting: Vercel, Netlify, GitHub Pages, Cloudflare Pages.
